Place Inter font files here for server-side (FFmpeg) export overlays.

The server looks for these filenames (first match wins):
  Regular: Inter_18pt-Regular.ttf or Inter-Regular.ttf
  Bold:    Inter_18pt-Bold.ttf or Inter-Bold.ttf
  Thin:    Inter_18pt-Thin.ttf or Inter-Thin.ttf

Use the same .ttf files you put in public/fonts/ for the frontend.
Restart the server after adding fonts.
